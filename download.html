📖 安装指南

Windows 安装

# 1. 下载 New.js-Setup-2.0.0.exe 安装包
# 2. 双击运行安装程序
# 3. 按照安装向导完成安装
# 4. 安装完成后，打开命令提示符或 PowerShell
# 5. 验证安装是否成功
newjs --version

# 创建新项目
newjs create my-app

# 运行项目
cd my-app
newjs run

Linux 安装

# 方法一：使用 wget 下载并安装
wget https://github.com/deeplearning-corporation/node-js/releases/download/v2.0.0/New.js-2.0.0.tar.gz
tar -xzf New.js-2.0.0.tar.gz
cd New.js-2.0.0
sudo ./install.sh

# 方法二：使用 curl 下载
curl -L https://github.com/deeplearning-corporation/node-js/releases/download/v2.0.0/New.js-2.0.0.tar.gz | tar xz
cd New.js-2.0.0
sudo make install

# 验证安装
newjs --version

# 创建新项目
newjs create my-app
cd my-app
newjs run

使用 npm 安装（所有平台）

# 全局安装 New.js
npm install -g newjs

# 创建新项目
newjs create my-app

# 进入项目目录
cd my-app

# 安装依赖
npm install

# 运行项目
newjs run

# 或使用 npm
npm start

从源码构建

# 克隆仓库
git clone https://github.com/deeplearning-corporation/node-js.git
cd node-js

# 安装依赖
npm install

# 构建
npm run build

# 链接到全局
npm link

💻 系统要求

Windows

• Windows 10 (版本 1809 或更高) / Windows 11
• Windows Server 2019 / 2022
• 64位处理器
• 至少 2GB RAM
• 500MB 可用磁盘空间

Linux

• Ubuntu 20.04 / 22.04 / 24.04
• Debian 11 / 12
• CentOS 8 / Stream 9
• RHEL 8 / 9
• Fedora 38 / 39
• 64位处理器
• 至少 1GB RAM
• 300MB 可用磁盘空间

通用要求

• Node.js 16.0 或更高版本（如果使用npm安装）
• npm 7.0 或更高版本
• 支持 ES6+ 的现代浏览器（用于开发）
